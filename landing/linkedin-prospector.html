Skill spotlight

LinkedIn prospecting, but with a real browser and an actual workflow.

RethinkSoft lets your AI agent search LinkedIn, read posts, inspect profiles, and draft personalized connection notes using your signed-in session instead of brittle scraping or guesswork.

Use this skill Build something similar
  • Find people by role, company, topic, or recent posting behavior.
  • Read activity before drafting outreach so messages feel contextual.
  • Keep humans in control of the final send while AI handles the heavy lifting.
Why this works

LinkedIn is a browser-native workflow.

This job depends on a logged-in UI, real navigation, and visual context. That is exactly the kind of task where browser automation gives an agent real leverage instead of forcing awkward workarounds.

Search intelligently

Start from a niche, location, role, or audience signal and let the agent explore the results pages like a human operator would.

Read before writing

Use recent activity and profile context so the connection request feels earned rather than generic.

Keep a repeatable system

Bundle the flow as a skill so the best prompt structure and browser behavior stay consistent across runs.

Install

Run the skill from your agent.

Use the base setup command first, then invoke the LinkedIn workflow through the browser CLI or from your connected coding agent.

Setup and run 
npx rethinksoft-in-chrome setup
rethinksoft-browser start "Find 10 healthcare recruiters in New York and draft tailored connection requests" --skill linkedin-prospector

That flow installs the browser layer, connects your agents, and starts a LinkedIn-specific task with reusable guidance.

Good prompts

  • 1State the niche or job title you are targeting.
  • 2Include geography or company filters when relevant.
  • 3Specify whether you want notes drafted, reviewed, or sent.
Flow

How the skill behaves.

The goal is not “click everywhere.” It is a calm, structured flow that searches, evaluates, personalizes, and only acts where the browser is genuinely needed.

1. Prospect discovery

Open LinkedIn search or a relevant feed and gather a candidate set from the live UI.

2. Context gathering

Read profiles and recent posts so outreach references something real and current.

3. Action with review

Draft a tailored note and let the human decide whether to send, revise, or skip.
