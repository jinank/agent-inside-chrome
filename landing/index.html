Build with the API

Add browser automation to your own product.

Your app can send a user a pairing link, wait for their browser to connect, run a task, and show the answer back in your own interface. The browser runtime becomes part of your product instead of a manual support workflow.

Flow
  • 1Create a pairing token for your user.
  • 2Show a pairing link so they connect their browser.
  • 3Run tasks against the connected browser session.
  • 4Poll for completion and render the answer inside your app.
import { HanziClient } from "@hanzi-browse/sdk";

const client = new HanziClient({
  apiKey: process.env.HANZI_API_KEY,
});

const { pairingToken } = await client.createPairingToken();
const pairUrl = `https://api.hanzilla.co/pair/${pairingToken}`;

const sessions = await client.listSessions();
const browser = sessions.find((session) => session.status === "connected");

const result = await client.runTask({
  browserSessionId: browser.id,
  task: "Read the patient chart on this page",
});

console.log(result.answer);
