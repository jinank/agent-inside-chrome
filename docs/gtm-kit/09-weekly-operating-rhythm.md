# Weekly Operating Rhythm

## Monday: Pipeline build + campaign prep

- Finalize target account list for the week
- Personalize outreach angles by segment
- Prepare 1 short demo artifact for current wedge use case

## Tuesday-Thursday: Outbound + discovery

- Send daily outbound batch
- Run discovery calls
- Update CRM/sheet immediately after each call
- Publish one micro-proof artifact (clip, screenshot, short writeup)

## Friday: Conversion + review

- Push active opportunities toward pilot decision
- Send pilot scope docs for qualified accounts
- Review weekly metrics and identify the bottleneck

## Weekly KPI review questions

- Is reply rate improving?
- Are meetings converting into pilots?
- Are pilots reaching measurable outcomes?
- Where is the main friction: message, fit, or delivery?

## Monthly reset

- Keep one ICP if conversion is healthy
- Narrow wedge if conversion is weak
- Promote strongest pilot into case study + outbound proof
