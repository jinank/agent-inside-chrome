# Offer and Pilot Design

## Flagship offer

"We automate one browser-critical workflow in 14 days."

## Offer components

- Workflow mapping session (60 min)
- Baseline measurement (time, errors, throughput)
- Pilot implementation
- Team enablement walkthrough
- ROI summary + expansion roadmap

## Pilot scope template

- Workflow name:
- Current owner:
- Current frequency:
- Current manual effort per run:
- Target automation level:
- In-scope systems:
- Out-of-scope systems:

## Success criteria template

- Reduce per-run effort by: `X%`
- Reduce total weekly hours by: `Y hours`
- Improve turnaround time by: `Z%`
- Maintain/Improve output quality baseline

## Pilot timeline (2 weeks)

1. Week 1:
   - Discovery
   - Baseline metrics capture
   - Technical implementation
2. Week 2:
   - Validation runs
   - Team handoff
   - Outcome readout and expansion proposal

## Expansion triggers

- Same workflow across other teams
- Additional browser-dependent workflows
- API integration into product experience

## Commercial framing options

- Paid pilot (recommended)
- Free pilot with strict scope + mandatory case study rights
